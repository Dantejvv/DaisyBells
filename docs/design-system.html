SWIFTUI MAPPING NOTES

Display font → use .font(.system(.largeTitle, design: .rounded, weight: .black)) or a custom condensed font

Colors → define in Assets.xcassets as named color sets with dark/light variants

Spacing → create CGFloat extension with .spacingSm, .spacingMd, etc.

Semantic colors → wrap in Color extension for .accent, .destructive, etc.
